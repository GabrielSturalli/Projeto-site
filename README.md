# Gabriel Sturalli — Portfólio Profissional

Portfólio digital de Gabriel Sturalli, economista em formação pela FECAP com foco em dados, ciência de dados e automação. Especialista em tesouraria e ALM com experiência em análise macroeconômica e desenvolvimento de dashboards interativos.

## 🌟 Destaques

- **Homepage profissional** com apresentação de CV e links para projetos
- **Blog** com análises de economia e comentários sobre fatores relevantes
- **Seção de vídeos** com 9 vídeos do canal Entendendo Dinheiro
- **Dashboard de indicadores macroeconômicos** com dados ao vivo do BCB e IBGE
- **Simulador de cenários** para curva de juros (taxa, inclinação, curvatura)
- **Automação de prospecção CNPJ** com filtro de 150+ empresas
- **6 trabalhos acadêmicos** em economia, dados e análise
- **Currículo interativo** com download direto
- **Design responsivo** com tema dark profissional

## 📂 Estrutura

```
├── index.html                          # Página principal
├── curriculum.html                     # Currículo profissional
├── blog.html                          # Blog de economia
├── videos.html                        # Galeria de vídeos
├── blog-posts.js                      # Dados dos posts
├── videos-data.js                     # Dados dos vídeos
├── curva-juros.js                     # Dados da curva ao vivo
├── blog-img/                          # Imagens de capas
├── [estudos]/                         # Páginas de trabalhos acadêmicos

└── README.md
```
